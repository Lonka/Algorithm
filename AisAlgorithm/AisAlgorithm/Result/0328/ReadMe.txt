訓練80%
測試20%

使用的欄位
	相對總用電
	相對空調用電
	平均氣溫
	平均濕度
	下個15分鐘用電	


分群方法
	1. 依第一點，用漢明每欄相似度5%，不重復加群
	2. 依第一點，用Fuzzy漢明，不重復加群

預測
	1. 依群中心值用漢明每欄相似度10%找相似的群
	2. 該群的每一個點算出皮爾森系數
	3. 該群的每一點加總 [皮爾森系數 * ( 該群點的下個15分鐘用電 - 預測筆的相對總用電 )]
	4. 該群的調整量為  3.加總值 / 該群每點皮爾森系數的加總
	5. 預測筆的相對總用電 + 4.該群的調整量。
	6. 2-5重復做完每一群

MSE約等於1.3832

檔名
F_T_X_0.000
F=是否有用fuzzy來分類
T=在分群上是否有把Target納進來看
X=次數
0.000=該次結果的MSE
